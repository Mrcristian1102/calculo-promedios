## Calculador de Promedios

### ¿Qué hace?
Este script sirve para pedir el nombre de un alumno, ingresar 3 notas y que el programa te diga cuál es el promedio final.

### Lo que aprendí:
* Cómo pedir datos al usuario con `input`.
* Cómo convertir texto a números con `float`.
* Cómo hacer operaciones matemáticas básicas.
* Cómo usar GitHub para guardar mi código.
